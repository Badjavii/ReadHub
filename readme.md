# ReadHub: Mercado Anónimo de Libros

## Introducción

**ReadHub** es una plataforma digital creada para la comunidad fanática de los libros.  
Su misión es ofrecer un **espacio seguro, confiable y eficiente** donde estudiantes puedan **comprar y vender libros** de manera sencilla.

---

## Funcionalidades Clave

La aplicación está diseñada para ser **intuitiva, rápida y poderosa**:

- **Acceso como visitante:** Explora el catálogo completo sin necesidad de registrarte.  
- **Registro de usuario:** Permite adquirir libros, consultar historial de compras y publicar libros en venta.
- **Búsqueda avanzada:** Encuentra libros por título, autor o género.  

> Nuestra prioridad es garantizar un software **estable, seguro y de alta calidad**, que siente las bases para futuras innovaciones.

---

## Tecnologías Empleadas

ReadHub sigue una arquitectura **Cliente-Servidor (Frontend/Backend)**, con herramientas modernas y escalables:

### Frontend
- **Vue.js** → Framework progresivo de JavaScript para construir interfaces reactivas y dinámicas.  

### Backend
- **ASP.NET Core (C#)** → Framework robusto para lógica de negocio y APIs RESTful.  

### Gestión y Persistencia
- **Git & GitHub** → Control de versiones y colaboración en equipo.  
- **JSON** → Formato estándar para intercambio de datos entre cliente y servidor.  

---

## Arquitectura del Proyecto

La arquitectura está basada en un enfoque modular y escalable:

```bash
src/
├── backend/     # Lógica del servidor: controladores, modelos, servicios, repositorios
└── frontend/    # Interfaz de usuario: componentes Vue, servicios, recursos estáticos
```

Ambos módulos se comunican mediante peticiones HTTP y JSON, siguiendo el estilo RESTful.
